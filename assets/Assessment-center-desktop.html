Ассесмент центр

Методика поиска эффективного работника среди кандидатов и сотрудников

Ассессмент-центр (Assessment center или Центр оценки) – это один из самых точных методов оценки компетенций. Он представляет собой набор деловых упражнений, имитирующих реальную деятельность сотрудников. Участники выполняют задания упражнения под наблюдением экспертов, которые оценивают участников по компетенциям. Ассессмент-центр может включать в себя: бизнес-кейсы, тесты, выполнение проектов в составе группы, деловые, ролевые игры.

Ассессмент-центр помогает понять людей, работающих в компании. Наиболее популярные компетенции для оценки:

  • -Понимание бизнеса;
  • -Стратегичность;
  • -Принятие решений;
  • -Работа в команде;
  • -Постановка задач и контроль (Управление исполнением);
  • -Ориентация на результат;
  • -Планирование;
  • -Навыки продаж;
  • -Влияние;
  • -Саморазвитие и наставничество.

Ассессмент-центр подходит для оценки, как действующих сотрудников, так и кандидатов. С ним вы снижаете риск ошибки взять не того человека. Этот метод в среднем дает лишь 25% ошибок, что в два раза точнее, чем интервью и тесты.

Внедрение ассессмент-центра позволит:

  • -Объективно оценить профессиональный уровень;
  • -Составить индивидуальный план развития для каждого участника;
  • -Определить степень соответствия должности;
  • -Выявить перспективных сотрудников для их дальнейшего развития и карьерного роста;
  • -Отобрать подходящих кандидатов на руководящие должности и ответственные позиции;
  • -Выявить особенности поведенческих стилей и компетенций участников;
  • -Подобрать кадры на вакантные должности по значимым для компании критериям;
  • -Спрогнозировать успешность подобранного специалиста на новом рабочем месте;
  • -Сформировать кадровый резерв, провести кадровые перестановки;
  • -Запустить новые проекты компании, ускорить ее развитие.

В результате руководители получают:

  • -Отчет по каждому участнику;
  • -Индивидуальные планы развития для каждого сотрудника;
  • -Общий отчет по сотрудникам;
  • -Матрицу кадрового резерва;
  • -Рекомендации по дальнейшему развитию.

По итогам проведения ассессмент-центра можно:

  • Сформировать кадровый резерв;
  • Составить систему кадровых лестниц;
  • Составить план обучения как для всей Компании, на основании того, какие компетенции получили наименьшие оценки, так и отдельно для каждого сотрудника;
  • Выявить группу сотрудников в развитие и обучение, которых не выгодно вкладывать деньги.
  • Существует несколько разновидностей ассессмент-центра. Выбор будет зависеть от ваших конкретных задач.

1 – первое место по заданному критерию, 4 – последнее

Примеры отчетов: отчет 1, отчет 2, отчет 3, отчет 4.
